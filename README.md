# Consolidador-de-Planilhas
A spreadsheet aggregator using React and Javascript. Basically, it takes four different spreadsheets containing information and data, whether they are different or the same, and retrieves that data with specific modifications to a final spreadsheet with this 

----------------------------

1. Install Node.js

Download the LTS version from: https://nodejs.org
Run the installer (Next → Next → Finish).

To verify the installation, open Command Prompt (Win + R → cmd) and run:

node -v

You should see something like v20.x.x.

2. Open the Project Folder in Terminal

Extract the .zip file.

Inside the project folder:

Click on the address bar in Windows Explorer

Delete the current path

Type cmd and press Enter

This will open the terminal directly in the project folder.

3. Install Dependencies (Only Once)

npm install

This may take 2–5 minutes and will create a node_modules folder.

4. Run the Project

npm run dev

You should see something like:

Local: http://localhost:3000

Open this link in your browser, the project will be running.

5. Stop the Project

Press:
Ctrl + C in terminal
